It's time for a fresh look :)

Checkout the repo and serve locally: 

      > git clone https://github.com/ImmunHelden/ImmunHelden.de
      > cd ImmunHelden.de
      > git checkout fresh
      > firebase serve --only functions,hosting:fresh
      > xdg-open http://localhost:5000

Deploy the repo to https://immunhelden-fresh.web.app/: 

      > firebase deploy --only hosting:fresh
